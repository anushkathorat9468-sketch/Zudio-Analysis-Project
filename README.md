# Zudio Sales Analysis

** Project Overview
This project analyzes Zudio sales data using Python to identify revenue trends, customer behavior, and factors affecting store performance.

** Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

** Key Objectives
- Analyze sales performance
- Compare revenue across categories
- Study the impact of staff count
- Evaluate store characteristics

** Key Insights
- Women's category generated significant revenue.
- Staff count showed a relationship with sales performance.
- Store features influenced revenue patterns.

- Zudio_Analysis.ipynb
- Dataset.csv
- README.md
